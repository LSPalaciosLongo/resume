# Luciano Palacios Longo - Online Resume

Personal online resume focused on data analysis, SAP reporting, KPI monitoring and industrial operations context.

## Stack

- HTML
- CSS
- Vanilla JavaScript

## Purpose

This site is designed to present Luciano's professional profile for Data Analyst, BI / Reporting Analyst and SAP-related roles with a cleaner recruiter-facing structure than a traditional PDF alone.

## Current sections

- Hero and positioning
- Professional summary
- Technical toolkit
- Work experience
- Education and certifications
- Proof-of-fit work themes

## Local usage

Open `index.html` in a browser.
